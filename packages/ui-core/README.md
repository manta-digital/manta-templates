# @manta/ui-core

React UI primitives and semantic tokens for Manta templates.

- Import components and utilities:
  - `import { Button, Card, BaseCard, cardVariants, cn } from '@manta/ui-core'`
- Import default tokens/styles in your app CSS:
  - `@import '@manta/ui-core/styles.css';`

Builds ESM + types. Expects Tailwind v4 in the consumer.
